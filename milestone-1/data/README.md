# Milestone 1 PDF 候选资料

本目录用于存放第一阶段里程碑验证所需的中文 PDF 文档。

## 当前候选

### 1. `rag_intro_aliyun.pdf`
- 来源：https://developer.aliyun.com/article/1685288
- 主题：RAG 入门
- 适合原因：
  - 最贴合当前 W4 主线
  - 结构完整，覆盖为什么需要 RAG、基本流程、向量检索、简易实现
  - 适合作为 `milestone_rag_pdf.py` 的第一份主文档

### 2. `hnsw_intro_cn.pdf`
- 来源：https://www.luxiangdong.com/2023/11/06/hnsw/
- 主题：HNSW / 向量检索索引原理
- 适合原因：
  - 对应 W3 的补强阅读
  - 适合设计“为什么暴力检索慢”“ANN 为什么可接受”“HNSW 为什么更快”这类问题
  - 篇幅较长，适合作为进阶问答材料

### 3. `prompt_engineering_cn.pdf`
- 来源：https://help.aliyun.com/document_detail/2990893.html
- 主题：Prompt Engineering 实践
- 适合原因：
  - 对应 W4 上半段 Prompt 基础
  - 包含角色设定、约束、示例、迭代优化等内容
  - 可作为 Prompt 设计和调优的补充材料

## 当前建议

- 如果只选一份做里程碑主文档：优先用 `rag_intro_aliyun.pdf`
- 如果做主线 + 补强：再加 `hnsw_intro_cn.pdf`
- 如果想顺手练 Prompt 相关问答：再加 `prompt_engineering_cn.pdf`
